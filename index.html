Basics Of The Web
- The web is a collection of HTML documents.
- HTML stands for hypertext markup language
-- Text is what it is displayed
-- Markup is how it is displayed (formatting)
-- HTML contains hyperlinks which are links to other content on the web. Hyperlinks make the web 'weblike'

HTML Basics
As stated previously, the web is a collection of HTML pages which can contain a wide variety of different content. A few examples of content are:
- text
- images
- video
- documents

HTML is constructed of various tags. Some tags have opening and closing tags and other tags do not have a closing tag. Tags without a closing tag are known as void tags.
Tags can be inline or block tags. An inline tag is just like it sounds. It goes inline with the text. Block tags have height and width. There is an invisible rectangle around block elements.

Computers Are Stupid
And Programmers Don't Need To Remember Everything
Computers only do exactly what the programmer instructs it to do. Very simple mistakes that a human could easily spot cause problems for computers. The code needs to be perfect with all tags properly constructed for the page to work properly.
There is alot for the programmer to remember to get the code correct. It is important that the programmer understand concepts, but the details can be found with a quick web search so they don't need to remember every detail.
